## 任务2：ESG报告/年报整理与文本预处理 【经济统计A】

### 2.1 已有数据
- 位置：`data_competition/data1/cninfo_reports/巨潮年报或ESG_2019-2021/`
- 已下载：~97份PDF（覆盖钢铁、建材、发电等行业）
- 目录结构按"行业/组别/企业名"组织

### 2.2 执行步骤

**Step 1：核对覆盖率**
```powershell
# 运行检查脚本
conda run -n data_competition python -c "
import os, pandas as pd
df = pd.read_excel('核验结果/最终企业样本名单.xlsx', engine='openpyxl')
data = df[df.iloc[:,3].astype(str).str.match(r'^\d{6}$')]
codes = [str(int(row.iloc[3])).zfill(6) for _, row in data.iterrows()]
names = [str(row.iloc[4])[:15] for _, row in data.iterrows()]

base = r'data_competition\data1\cninfo_reports\巨潮年报或ESG_2019-2021'
missing = []
for i, (code, name) in enumerate(zip(codes, names)):
    found = False
    for root, dirs, files in os.walk(base):
        if code in root:
            pdfs = [f for f in files if f.endswith('.pdf')]
            if len(pdfs) >= 3:
                found = True
                break
    if not found:
        missing.append(f'{code} {name}')
print(f'Total: {len(codes)}, Missing: {len(missing)}')
if missing:
    for m in missing: print(f'  {m}')
"
```

### 3.2 模型选择
推荐使用哈工大讯飞中文RoBERTa：
- 模型名：`hfl/chinese-roberta-wwm-ext`
- 备选：`bert-base-chinese`（更轻量）

### 3.3 测试代码（先用1-2家企业试跑）
```python
# test_roberta.py
from transformers import AutoTokenizer, AutoModelForSequenceClassification
import torch

model_name = "hfl/chinese-roberta-wwm-ext"
tokenizer = AutoTokenizer.from_pretrained(model_name)

# 测试文本（从ESG报告截取）
sample_texts = [
    "公司将加大环保投入，计划2025年实现碳中和目标。",   # 承诺类
    "2021年公司实际投入环保资金1.2亿元，建成污水处理设施3座。",  # 行动类
    "本集团高度重视可持续发展，积极履行社会责任。",  # 中性/模糊
]

inputs = tokenizer(sample_texts, padding=True, truncation=True, return_tensors="pt")
print(f"Tokenized: {inputs['input_ids'].shape}")
```

### 3.4 文本特征提取Pipeline设计
目标：将ESG报告文本转化为 **承诺-行动分类** 特征

```
ESG报告PDF 
  → pdfplumber提取文本 
  → 按段落分割（每段≤512 tokens） 
  → RoBERTa编码 → [CLS]向量 
  → 二分类：承诺(1) vs 行动(0) 
  → 每个企业计算"承诺兑现比" = 行动段数/(承诺段数+行动段数)
```

### 3.5 关键指标
- **承诺兑现比（CAR）**：行动语句数 / (承诺语句数 + 行动语句数)
  - CAR > 0.6：言行一致（执行力强）
  - CAR 0.3-0.6：中等
  - CAR < 0.3：高漂绿嫌疑（说多做少）
